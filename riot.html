As a festival goer I felt that there weren't enough applications using a clear approach to list festivals around the globe. So I built an app using a Google API and MongoDB for the database.

Users login and can browse festivals via the map, giving them the ability to view according to the festival locations. It also features a weather API with daily updates; being unprepared for British summers is a thing of the past.
